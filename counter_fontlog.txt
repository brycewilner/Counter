Fontlog for Counter

This file provides detailed information on the Counter font software. This information should be distributed along with the Counter font and any derivative works.

Basic Font Information

Counter is a Unicode typeface family that supports all languages that use the Latin script and its variants, and could be expanded to support other scripts.
